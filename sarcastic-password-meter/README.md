# Sarcastic Password Strength Meter

A novelty single-utility web app that scores passwords and gives sarcastic feedback.

## Visual style

The app uses a **90s GeoCities Cyber Edition** theme: neon gradients, chrome-like panels, glowing text, and playful UI flair.

## Run locally

This is a static HTML/CSS/JS app.

Option 1 (quick):
1. Open `index.html` directly in a browser.

Option 2 (recommended local server):
1. From this directory, run:
   - `python3 -m http.server 8000`
2. Open:
   - `http://localhost:8000`

## Features

- Live password scoring (0 to 100)
- Dynamic progress bar with color levels
- Checklist for common password quality rules
- Show/Hide password toggle
- Contextual sarcastic feedback text
