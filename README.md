# IFT565_WEB_APP

Nurburgring Lap Predictor is a static HTML/CSS/JS web app where users pick popular cars, tweak specs, and get an estimated Nordschleife lap time based on historical reference runs.

## Run locally

Because this app is fully static, you can open `index.html` directly, or serve the folder with a tiny web server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
